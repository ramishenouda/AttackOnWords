### Prerequisites

- Unity

```
https://unity3d.com/get-unity/download
```

### Installing

1. Clone the repo.

```
git clone https://github.com/ramishenouda/AttackOnWords.git
```

2. Open the project using unity

3. Run the project using Unity.

```
Enjoy
```
